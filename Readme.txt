1 プログラムの動作
本プログラムは入力された文章をネットワーク文法に基づき形態素解析するものである。
1.1 実行環境
OS:Linux
1.2 使用方法
本プログラムのソースコードが格納されたディレクトリ内で try 〇.sh というシェルスクリプト
を実行する。tryC.sh はinput1.txt、tryG.sh ではinput2.txt の入力が実行される。
2 データの概要
2.1 入力データ
•input.txt…解析したい文章。
•dictionary.txt…単語辞書。単語と品詞の対。
•relation.txt…文法規則。品詞の対。
2.2 出力データ
result.txt とコンソール上に解析結果が出力される。解析した文章が文法規則に沿っていた場合
は correct、沿っていなかった場合は規則と異なっていた場所で処理が中断され、末尾に incorrect
と表示される。

3 モジュール
3.1 define.h
各モジュールで使用する構造体や定数について定義する。
3.2 readDictionary.c
単語辞書を読み込む。
3.3 check.c
読み込んだ単語辞書から入力文の単語の品詞を判別する。文法規則を読み込み、入力文の各単語
の品詞のつながりが文法規則に沿っているかチェックする。
3.4 main.c
入力文を上記の関数を用いて処理し、逐次結果を出力する。
